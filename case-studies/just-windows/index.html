The Problem

Just Windows had a working website — but it was costing them money, visibility, and leads every day it stayed online.

The old site

  • WordPress/PHP engine requiring a live server, database, and ongoing plugin updates
  • 20+ external scripts loading on every page — slowing down mobile visitors
  • 7 pages with no H1 heading — Google couldn't determine what each page was about
  • 8 pages where content was buried in plugin-generated markup with a low text-to-HTML ratio
  • 40 resources incorrectly formatted as page links — confusing search crawlers
  • Out-of-area keywords targeting cities Just Windows doesn't serve
  • Quote form with no automated response — leads going cold while replies were drafted manually

What they needed

  • A site that loads instantly on mobile — home services buyers search on their phones
  • Clean local SEO targeting Mentor, OH specifically, not diluted with irrelevant geography
  • Hosting that doesn't need a developer on call to stay online
  • Every page readable by Google — proper headings, content in plain HTML, structured data
  • Automated lead response so no quote request sits unanswered while competitors reply first

What ClearPath Built

Website Rebuild

  • 7 pages rebuilt as clean semantic HTML — no server, no database, no plugins required
  • One stylesheet and one script file replace 20+ external plugin scripts loaded by the old site
  • Homepage video uses preload="metadata" — doesn't download until the visitor chooses to play it
  • All images and assets hosted locally — no third-party CDN dependency on page load
  • Mobile-responsive navigation, layouts, and quote request form
  • Deployed to Azure Static Web Apps with a GitHub Actions CI/CD pipeline

SEO Optimization

  • Keywords researched in Semrush and mapped to the correct page by search intent — not stacked on every page, which dilutes relevance
  • Unique title tag and meta description on every one of the 7 pages
  • One keyword-optimized H1 per page — previously 7 of the pages had none at all
  • LocalBusiness structured data — tells Google this is a Mentor, OH home-services business
  • AggregateRating schema using the real 5.0/12 review data — eligible for Google star display in search results
  • sitemap.xml, robots.txt, canonical URLs, and Azure clean-URL rewrites (/windows not /windows.html)
  • Out-of-area keywords intentionally removed — ranking for the actual service area converts better than broad geographic targeting

Lead Automation — Power Automate

  • Power Automate flow triggers the moment a visitor submits the quote form on the site
  • Customer immediately receives a confirmation email — sets expectations and signals professionalism
  • Owner receives an instant alert with the lead's details — no manual inbox monitoring required
  • According to research by InsideSales and MIT, leads are 21× more likely to qualify when contacted within 5 minutes vs. waiting 30 minutes — the automation makes that response window automatic
